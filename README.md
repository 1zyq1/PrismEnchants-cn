# PrismEnchants-中文

PrismEnchants 的**完整中文本地化版本**，内置 55+ 种自定义附魔，支持附魔师商店 GUI、铁砧附魔、修复师等全套功能。

![Minecraft](https://img.shields.io/badge/Minecraft-1.21.4-blue)
![Spigot](https://img.shields.io/badge/Spigot-1.21.4-green)

## 特性

- **55+ 种自定义附魔** — 涵盖剑、弓、工具、盔甲等
- **附魔师商店 GUI** — 在游戏内 GUI 消耗经验等级获取附魔书
- **修复师** — 出售附魔物品返还经验点数
- **完整中文** — 所有附魔名称、描述、GUI 界面、消息均为中文
- **公平经验系统** — 扣费基于固定经验点数而非等级（高等级不亏）

## 附魔师商店（GUI）

游戏内输入 `/pe` 打开附魔师商店。消耗经验等级随机获得对应稀有度的附魔书：

| 选项 | 消耗 | 可获得稀有度 |
|------|------|------------|
| 学徒 | 15 级 | 普通 | 不凡 |
| 大师 | 30 级 | 稀有 | 史诗 |
| 传奇 | 45 级 | 传说 | 神话 |
| 白卷轴 | 10 级 | 重置附魔 |
| 黑卷轴 | 8 级 | 清除附魔 |

> 消耗基于**固定经验点数**（而非等级），无论玩家多少级，每次消耗的实际经验点相同。

## 修复师

将附魔物品放入修复师 GUI，出售返还 **85%** 经验点数（以经验点数而非等级计算，公平统一）。

## 安装

1. 将 `PrismEnchants-中文.jar` 放入服务器的 `plugins/` 目录
2. 重启服务器或 `/reload`
3. 初始配置会自动生成在 `plugins/PrismEnchants/config.yml`

## 命令

| 命令 | 描述 |
|------|------|
| `/pe` | 打开附魔师商店 |
| `/prismenchants give <玩家> <附魔ID> [等级]` | 给予附魔书 |
| `/prismenchants reload` | 重载配置 |

## 配置

`config.yml`：
```yaml
# 附魔师商店成本 (XP等级)
enchanter:
  apprentice-cost: 15
  master-cost: 30
  legendary-cost: 45
  white-scroll-cost: 10
  black-scroll-cost: 8
```

`enchants.yml`（自动生成）— 每项附魔可独立开关、调整成功率和摧毁率。

## 技术说明

- 基于原版 PrismEnchants，进行完整中文翻译（附魔名称、描述、GUI、消息）
- 经验消耗改为固定经验点数（遵循 MC 1.21.4 经验公式），高等级玩家不会多扣
- 繁荣附魔有玩家方块放置检测，防止刷物品漏洞
- 支持 Vault 经济系统（可选依赖）
